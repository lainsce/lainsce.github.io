Workflow for the Day 19O04

Every day, I try to follow a strict program almost like a robot, but this programming is good for what I do, which is code at home. I always try to do at least one thing, and I have done so since 19D07. It's been a strange yet fulfilling journey. I also make sure to log it into a (big by now) text file which is structured like a Markdown table, with the task code for the day, the day task, and how many pulses were used for that task. An example: 

DATE  CODE  DESC                  PS
19O04 1A    Logged a naxasero     7
  *   1B    Clark New Album       2
  *   2A    Write parser for this 15
  *   1A    Logged a naxasero     7
As you can see, every task has a code. To understand my codes, you need to understand that I divide things into three sections: Leisure, Work, and Research. Each of those is a number; respectively, 1, 2, and 3. A Table for each level and code follows:

Leisure (1) Work (2) Research (3)
A - Text A - Code A - Read
B - Audio B - Build B - Listen
C - Video C - Design C - Explore
D - Idle D - Idle D - Idle
That's it. So, a 1A code corresponds to a leisure activity about text. I also make sure to diversify tasks so that no task takes all the day, even if it should. The amount of pulses also is a deciding factor about whether I should move on, as if a task exceeds 30 pulses on the day, it's time to take a breath.
