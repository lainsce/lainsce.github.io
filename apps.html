Coin

$ 1.00

Vala

Cryptocurrency

Tracking

Never miss out on the value of a virtual currency with this applet

  • Choose which currencies to track
  • Quit anytime with ⌘ Q
  • Move the applet by dragging it from anywhere in the window
  • Stays out of your way in the desktop

Feeling adventurous? Start by cloning the repo, then compiling it locally:


                    $ meson build && cd build
                    $ mesonconf -Dprefix=/usr
                    $ sudo ninja install

If you have any new features or you've fixed a bug locally, don't forget to submit a Pull Request on GitHub.
